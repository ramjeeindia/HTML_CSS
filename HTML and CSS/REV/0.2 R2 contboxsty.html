The box-sizing CSS property sets how the total width and height of an element is calculated.

content-box : Content size Fixed but BOX size will OUTSIDE to his Parent container : Parent container=600px , Child container=200px , Border 1+1 =2px , Padding 5+5 =10px then Content+Box Size = 200+2+10=222 px

BORDER BOX ELEMENT-WIDTH = Child container-WIDTH - ALL BORDER-WIDTH - ALL PADDING

EW = CW 200px + B (1px LEFT + 1 px RIGHT) + P (5px LEFT + 5 px RIGHT) = Total 222px
