align-items define TOP to BOTTOM position if container have height

The align-items property specifies the default alignment for items inside a flexbox or grid container.In a flexbox container, the flexbox items are aligned on the cross axis, which is vertical by default (opposite of flex-direction). In a grid container, the grid items are aligned in the block direction. For pages in English, block direction is downward and inline direction is left to right.
