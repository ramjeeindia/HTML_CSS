Introduction to HTML

HTML stands for Hyper Text Markup Language . It is a markup language used to create web pages and web applications.
Let's understand the HTML definition terms. Create a new file with the name "index.html" then type ! then press tab or Enter

Different parts of the HTML code

!DOCTYPE html: This is doctype declarion , which specifies the version of HTML being used in the document .In this case it's HTML5
html : This is the root element of the HTML document and it contains all the other elements in the document
head : This is nthe head section of the HTML documnet which contains meta-information about the document such as the title of the web page that appears in the browser's title bar or tab
title : This is the title tag which set the title of the web page
body : This is the body section of the HTML document which contains the visible content of the web pages such as headings, paragraphs and more
h1-h6 : this is the heading tag used to define a top level heading
